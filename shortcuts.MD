## Tableau des raccourcis VS Code

### Navigation générale
| Action | Raccourci |
|-------|-----------|
| Ouvrir la palette de commandes | Ctrl + Shift + P |
| Ouvrir la recherche de fichiers | Ctrl + P |
| Changer d’onglet | Ctrl + Tab |
| Fermer l’éditeur | Ctrl + W |
| Nouveau fichier | Ctrl + N |
| Ouvrir un dossier | Ctrl + K, puis Ctrl + O |

### Édition de code
| Action | Raccourci |
|--------|-----------|
| Aller à la définition | F12 |
| Déplacer une ligne | Alt + ↑ / ↓ |
| Copier une ligne | Shift + Alt + ↑ / ↓ |
| Sélectionner mot par mot | Ctrl + Shift + ← / → |
| Sélectionner la ligne | Ctrl + L |
| Sélectionner toutes les occurrences | Ctrl + Shift + L |
| Sélectionner l’occurrence suivante | Ctrl + D |

### Recherche & remplacement
| Action | Raccourci |
|--------|-----------|
| Rechercher dans le fichier | Ctrl + F |
| Remplacer dans le fichier | Ctrl + H |
| Recherche globale | Ctrl + Shift + F |
| Fermer la recherche globale | Ctrl + Shift + E |

### Dans le terminal
| Action | Raccourci |
|--------|-----------|
| Supprimer un mot | Ctrl + w |
| Supprimer une ligne | Ctrl + u |
